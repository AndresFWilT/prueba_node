# prueba_node
Prueba node.js, usando express, para un CRUD de una BD postgres
# instalacion
- se necesita node version 16.17.0
# dependencias
- cors: ^2.8.5,
- express: ^4.18.1,
- morgan: ^1.10.0,
- pg: ^8.7.3,
- reflect-metadata: ^0.1.13,
- typeorm: ^0.3.7
# ejecucion
El proyecto viene con el script start para su ejecucion, utilizar:
´´´
npm start
´´´
